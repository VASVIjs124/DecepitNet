# Examples Directory

This directory contains example scripts demonstrating DECEPTINET capabilities.

## Available Examples

### 1. Demo (`demo.py`)
Complete platform demonstration showing all components working together.

```bash
python examples/demo.py
```

### 2. Honeypot Example (`honeypot_example.py`)
Deploy a standalone SSH honeypot.

```bash
python examples/honeypot_example.py
```

### 3. Red Team Example (`redteam_example.py`)
Run autonomous red team simulation.

```bash
python examples/redteam_example.py
```

## Requirements

Make sure you've installed all dependencies:
```bash
pip install -r requirements.txt
```

And initialized the platform:
```bash
python main.py --init
```
